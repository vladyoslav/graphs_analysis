# SPLA — разреженная линейная алгебра

**SPLA** ([репозиторий](https://github.com/SparseLinearAlgebra/spla), [C++ API](https://SparseLinearAlgebra.github.io/spla/docs-cpp/)) — MIT, **разреженная линейная алгебра** на **GPU (OpenCL)** и **CPU**.

- **Производительность**: типовые ядра (**m×v**, редукции, маски) на GPU эффективнее ручных циклов по графу
- **Semiring**: одни и те же примитивы под разные задачи — свои `×` / `+` и типы
- **Портируемость**: OpenCL, выбор устройства; API **C/C++** и **Python (pyspla)**

---

# Apache Spark

**Apache Spark** ([spark.apache.org](https://spark.apache.org/)) — **распределённый** движок вычислений поверх кластера (**Scala**/**Java**/**Python**/**R**); ориентир на **память** и **план запросов** вместо лишних обращений к диску.

- **Масштаб и устойчивость**: разбиение данных по узлам, перезапуск задач при сбоях, рост почти «добавили воркеры»
- **Стек в одном месте**: **Spark SQL**, **DataFrame/Dataset**, потоки (**Structured Streaming**), **MLlib** — без сшивания десятка разных систем
- **Графы**: **GraphX** и супер-шаги в духе **Pregel** — итеративные обходы и агрегации по распределённому графу

# Алгоритм Борувки

**Задача**: найти **минимальное остовное дерево** (MST) взвешенного связного графа.

**Шаги одной итерации**:
1. Для каждой компоненты найти минимальное исходящее ребро
2. Добавить найденные рёбра в MST
3. Объединить компоненты, соединённые выбранными рёбрами

**Повторяем, пока не останется одна.**

---

# Борувка в терминах линейной алгебры

**Данные:**
- $S$ — матрица смежности: $S[i,j] = w(i,j)$
- $parent[v]$ — корень компоненты вершины $v$
- Ребро — кортеж $(w,\, i)$: $w$ — вес, $i$ — компонента соседа
- $edge[v]$ — мин. ребро из вершины $v$ в другую компоненту
- $cedge[v]$ — мин. ребро из компоненты $v$ наружу (только для корней, иначе $+\infty$)

**Полукольцо** $combMin$:
- $\otimes$: $comb\bigl((w_1, i_1),\, (w_2, i_2)\bigr) = (w_1,\, i_2)$ — взять вес из первого кортежа, индекс из второго
- $\oplus$: $min$ — выбрать кортеж с меньшим весом

---

# Борувка в терминах линейной алгебры

**Инициализация:** $parent[v] = v$

**while** $S \neq \emptyset$:

1. $edge = S \times_{combMin} parent$
   — для каждой вершины найти мин. ребро наружу через mxv
2. $cedge[parent[i]] = min\,(cedge[parent[i]],\; edge[i])$
   — для каждой компоненты выбрать мин. ребро из всех её вершин
3. Обновить $parent$: корни переходят к соседней компоненте, сжатие пути
4. $S = select(S,\; parent[i] \neq parent[j])$
   — удалить рёбра внутри компонент
